    Комментарии

    Элемент WebBrowser позволяет размещать веб-страницы и другие документы, поддерживаемые обозревателем, в приложениях Windows Forms. Можно использовать элемент WebBrowser, например, для предоставления встроенной пользовательской справки на основе HTML или возможности обзора интернета в приложении. Кроме того, можно использовать элемент управления WebBrowser, чтобы добавить существующие веб-элементы управления в клиентские приложения Windows Forms.

    Важно

    Элемент управления WebBrowser потребляет много ресурсов. Не забудьте вызвать Dispose() метод при завершении работы элемента управления, чтобы обеспечить освобождение всех ресурсов в течение отведенного времени. Необходимо вызвать метод Dispose() в потоке, присоединившем события, который всегда должен быть потоком сообщений или потоком пользовательского интерфейса (UI).

    WebBrowser Элемент управления не может использоваться частично доверенным кодом. Для получения дополнительной информации см. Using Libraries from Partially Trusted Code.

    WebBrowser Элемент управления имеет несколько свойств, методов и событий, связанных с навигации. Следующие элементы позволяют перемещать элемент управления по определенному URL-АДРЕСУ, перемещение назад и вперед по списку журнала переходов и загрузить домашнюю страницу и страницу поиска текущего пользователя.
